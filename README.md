# 清风正道 - 八项规定模拟游戏

一个基于中央八项规定的公务员模拟体验游戏，帮助玩家了解廉政纪律和正确的工作方式。

## 🎮 游戏特色

- **真实场景模拟**：基于真实公务员工作场景设计
- **八项规定学习**：深入理解中央八项规定精神
- **职业发展路径**：从科员到局长的完整仕途体验
- **成就系统**：多种成就等待解锁
- **案例库**：真实违纪违法案例警示教育

## 🚀 如何开始

1. 打开 `index.html` 文件
2. 点击"开始仕途"按钮
3. 在各种工作场景中做出选择
4. 遵守八项规定，实现职业发展

## 📋 游戏规则

- **群众满意度**：体现为民服务水平
- **城镇发展**：反映工作能力和成效
- **政治声望**：展现个人威信和影响力
- **廉政风险**：违规行为会增加风险值

## 🎯 获胜条件

- 保持较高的群众满意度
- 推动城镇发展进步
- 维护良好政治声望
- 控制廉政风险在安全范围

## ⚠️ 失败条件

- 廉政风险达到100（违纪违法被查处）
- 群众满意度降至0（脱离群众被免职）
- 政治声望降至0（被调离领导岗位）

## 🏆 职业等级

1. 科员 - 街道办事处
2. 副主任科员 - 街道办事处
3. 主任科员 - 街道办事处
4. 副科长 - 区民政局
5. 科长 - 区民政局
6. 副处长 - 市发改委
7. 处长 - 市发改委
8. 副局长 - 市政府办
9. 局长 - 市政府办

## 📚 教育意义

本游戏旨在：
- 普及中央八项规定知识
- 提高廉政意识
- 培养正确的工作作风
- 警示违纪违法行为的严重后果

## 🛠️ 技术实现

- 纯前端实现，无需服务器
- 使用 HTML5、CSS3、JavaScript
- 响应式设计，支持多种设备
